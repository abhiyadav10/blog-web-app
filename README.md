# Blog : Full Stack Web Application

>>This is a Blog application made from NodeJs,EJS,MongoDB and Basic Html,CSS,Javascript.
>>How to start the program

1. Create a .env file
2. Create your cluster on MongoDB Atlas and copy the link and paste in .env
   eg: MONGODB_URI = Yourlink of Mongo 
3. Also add in .env 
   eg: JWT_SECRET = MyTopSecretBlog
In Your terminal
4. Run: npm i
5. Run: npm run dev
